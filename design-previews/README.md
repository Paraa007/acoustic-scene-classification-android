# Design Previews

Static HTML mockups for the planned redesign of the Acoustic Scene
Classification app. Three directions, five screens each.

Open `index.html` in any browser — no build step, no server.

## The three directions

- **Lab Console** — `console.html` — dark and technical. The app as a
  measurement instrument: monospace readouts, a hairline grid, one
  signal-green accent.
- **Data Dashboard** — `dashboard.html` — dark and modular. A bento grid
  of tiles and charts, the nine scene colors used as a data palette.
- **Calm Focus** — `calm.html` — light and editorial. Warm paper,
  oversized serif headlines, one focus per screen.

Each file is self-contained: inline CSS, web fonts, inline SVG. No
dependencies.

## Screens

Dashboard · New Session (Step 1) · Live Recording · Session Results · History.

The wizard flow and the app's navigation are unchanged — this is a visual
redesign. Once a direction is picked, it gets implemented natively in the app.
